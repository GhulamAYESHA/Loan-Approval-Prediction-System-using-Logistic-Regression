# Loan-Approval-Prediction-System-using-Logistic-Regression

A machine learning-based Streamlit web application that predicts loan approval
using Logistic Regression.

## Features
- Data validation and preprocessing
- Model training and evaluation
- Explainable feature importance
- Interactive loan approval prediction
- Adjustable probability threshold

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
